DATE_GEN
========

Setup
-----

1. Run setup.py

	"""
	>python setup.py
	"""

2. Create the empty sqlite database:


	"""
	>python date_gen/database_interface.py
	"""

Run Program
-----------

For example,

	"""
	>python date_gen/date_gen.py --in T --s winter --n 2
	"""

To get help on option, do

	"""
	>python date_gen/date_gen.py --h
	"""
